# Hybrid AC/DC Microgrid — ODL-VC (v2)

## Key features
- 24 hourly profiles (renewable, two DC load buses, one AC load), compressed 24h→24s
- Import/export: P_grid = P_ren - (P_dc_loads + P_ac_load)
- Tight Vdc constraint band (±4 V) + tuned parameters
- Robustness notebook (10 kHz vs 15 kHz, ±30% Lf,Rf)

## Notebooks
- `notebooks/01_run_odlvc_24h_profiles.ipynb`
- `notebooks/02_robustness_10k_15k_pm30.ipynb`
